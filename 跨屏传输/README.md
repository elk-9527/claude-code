# 跨屏协同 · 使用说明

USB 有线、苹果风格的多屏协同工具。PC 鼠标键盘可穿越到安卓平板,实现键鼠跨屏操作。

## 交付物(在 `dist/` 目录)

| 文件 | 装在哪 | 说明 |
|------|--------|------|
| `跨屏协同-电脑.exe` | Windows 电脑 | 单文件绿色版,双击即用,无需安装 |
| `跨屏协同-平板.apk` | 安卓平板 | 安装包,约 5.6MB |

## 首次使用(三步)

### 第 1 步:平板装 APK 并授权
1. 把 `跨屏协同-平板.apk` 传到平板,点击安装(若提示「未知来源」,允许即可)
2. 打开 App,点两个按钮分别授权:
   - **开启无障碍服务** → 在列表里找到「跨屏协同」→ 打开开关(这是控制平板的关键)
   - **授予悬浮窗权限** → 打开开关(用于显示鼠标光标)
3. 两项都变成 ✓ 后,App 显示「等待 USB 连接…」

### 第 2 步:平板开启 USB 调试(只需一次)
1. 平板 **设置 → 关于平板**,连续点击「版本号」7 次,开启开发者选项
2. **设置 → 系统 → 开发者选项 → 打开「USB 调试」**

### 第 3 步:连接
1. 电脑双击 `跨屏协同-电脑.exe` 打开
2. 用 **USB 数据线**把平板连到电脑(平板会弹「允许 USB 调试」,勾选「一律允许」并确定)
3. 电脑端会列出平板设备,点击 → 点「连接」
4. 看到绿灯「已连接」即成功

## 怎么穿越控制平板

- **穿越过去**:鼠标移到电脑屏幕**最右边缘**,或按 `Ctrl + Alt + →`
- **控制**:在平板上移动光标、单击、长按、拖动滑动、滚轮滚动、键盘打字
- **系统键**:`Alt + ←` 返回 / `Alt + Home` 主页 / `Alt + ↑` 多任务
- **退回电脑**:鼠标移到屏幕**最左边缘**,或按 `Esc`

## 已知限制(MVP 阶段)

- **打字**只能输入到平板上**已聚焦的输入框**(无障碍服务的限制),中文需配合平板自身输入法
- 通过无障碍「手势」模拟点击,极少数对手势有防护的 App(如部分银行/游戏)可能不响应
- 目前一次连一台平板;断线后重新点「连接」即可
- 平板息屏会断开,亮屏后重连

## 给开发者:重新构建

### PC 端(需要 Node.js)
```bash
cd pc
npm install                    # 国内需设 ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/
npm start                      # 本地运行
npm run dist                   # 打包(需 ELECTRON_BUILDER_BINARIES_MIRROR 镜像)
```

### Android 端(需要 JDK 17+ 和 Android SDK)
```bash
cd android
# local.properties 里写好 sdk.dir
./gradlew :app:assembleDebug   # 产物在 app/build/outputs/apk/debug/
```

## 技术架构
```
PC(Electron)
  ├─ 主进程:adb forward 建 USB 隧道 + TCP 客户端
  ├─ 控制面板:苹果风 UI(设备列表/连接状态)
  └─ 穿越覆盖层:全屏透明窗口捕获鼠键 → 归一化坐标
        │  USB 数据线
        ▼  adb forward tcp ⇄ tcp:5566
平板(APK)
  ├─ BridgeService:前台服务,TCP 服务端 + 悬浮窗光标
  └─ InputAccessibilityService:无障碍注入(点击/滑动/输入)
```
协议细节见 [PROTOCOL.md](PROTOCOL.md)。
